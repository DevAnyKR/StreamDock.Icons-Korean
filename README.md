## 사용법

### Stream Dock 플러그인 번역 파일 설치 위치

#### 아이콘 ([user](https://github.com/DevAnyKR/StreamDock.Icons-Korean/tree/main/user) 폴더 내의 플러그인 폴더들)

`%AppData%\HotSpot\StreamDock\icons` 디렉터리에 기존 아이콘 폴더를 삭제 후 덮어씁니다.

### 프로그램 재실행

Stream Dock 프로그램을 종료 후 재시작하면 적용됩니다.
